# 项目总览（video-template-service）

## 这是什么

一个**轻量级模板化短视频生成服务**：用户上传素材、输入文字、选择固定模板，由**服务端**按模板自动完成拼接、叠加、画中画与文字排版，输出统一风格的成片链接。

产品重点是「把固定制作流程模板化」，而不是提供专业剪辑自由度。

## 这不是什么

- **不是**剪映 / Premiere / CapCut 类万能编辑器。
- **不是**复杂时间轴剪辑产品。
- **不是**浏览器端视频编解码实验场（也不优先采用 `ffmpeg.wasm`）。
- **不是**以 AI 视频生成为核心的项目（Round 0 更不接入真实大模型）。
- Round 0 **不包含**登录、支付、完备用户体系。

## 目标用户与场景

- **运营 / 自媒体**：批量产出固定栏目样式的短视频片头片尾与字幕样式一致的内容。
- **企业内部工具**：将「同一套视觉规范」的视频产出脚本化，减少手工剪辑重复劳动。

## 核心场景（未来 MVP）

1. 上传主素材（必要时上传画中画素材）。
2. 输入一小段说明文字。
3. 选择如「竖屏 1080×1920」模板。
4. 几分钟后得到合成视频下载或预览地址。

## MVP 范围（Roadmap 对齐）

- 固定少量模板；每层布局由 JSON 描述。
- 服务端 FFmpeg 流水线合成（后续里程碑）。
- 文案可由规则或轻量 AI 拆分标题、标签（可选）。

当前 **Round 0** 仅完成：**骨架、文档、Mock API、类型定义**，不接真实合成与外部 AI。
